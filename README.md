# 明日方舟 · 游戏设计深度拆解

> 一份面向游戏策划岗位的作品集分析——从商业化数值、玩法资源养成到平台期/成长期经济系统，系统性拆解这款上线逾六年仍保持稳定流水的二次元策略游戏。

**[→ 在线查看](https://your-username.github.io/arknights-analysis)**

## 内容结构

### 01 · 商业化数值基础
- 基于概率递增机制构建抽卡期望模型，量化 UP 角色获取成本
- 拆解卡池结构（限定池 / 联动池 / 常规双UP）与保底机制
- 分析皮肤定价体系（15–24 源石）及其对付费转化的影响

### 02 · 玩法 · 资源 · 养成线
- 构建玩法分层模型（登录层 / 活跃层 / 次要玩法），梳理玩家行为路径
- 拆解集成战略、卫戍协议等次要玩法的高留存机制
- 分类资源体系并分析资源循环，评估养成压力与长期稳定性

### 03 · 平台期 · 成长期经济
- 以周期视角分析产耗关系（时间轴产出 vs 体力产出）
- 体力系统横向对比：明日方舟 vs 崩坏：星穹铁道
- 构建成长期进度时间轴模型，分析玩家养成周期差异

## 数据说明

本分析中所有数值均基于以下来源，未有来源支撑的部分已明确标注：

| 数据类型 | 来源 |
|------|------|
| 理智系统参数 | 游戏内实测（85级，入坑约半年） |
| 精二养成理智消耗 | 明日方舟一图流（2500–3500 区间） |
| 成长期养成周期 | 月卡玩家实测，约 8 天（2025年9月版本） |
| 游戏时长分布 | 个人实测：日常 5–10 分钟，复刻活动约 2 小时，忠实玩家 3H+ |
| 卡池概率设计 | 官方公示概率说明 |
| 星穹铁道体力参数 | 游戏内公示数据 |
